Food Runners

Deployed App - Git Front-End - Git Back-End

React.js | Node.js | Express | Mongoose

Food Runners was created as a team project where I was responsible for designing the database architecture and UI.

The app allows busy food truck operators to receive orders from customers. The app then integrates a dashboard for the kitchen to allow for order fulfillment.

This specific implementation is for a pizza food truck where customers are able to sign-up for the service and customize ingredients before submitting the order to the food truck for preparation. The app also allows users to review the food truck.

Origin Marketplace

Deployed App - GitHub

JavaScript | HTML | CSS | Node.js | Express | MongoDB

Origin Marketplace allows coffee growers to list their coffee for direct trade in destination markets.

The app implements Google OAuth to authenticate growers. It then allows coffee growers to list their available coffees in the marketplace. Buyers can browse available coffees and see details such as notes, variety, process, and more.
