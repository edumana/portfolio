Food Runners

Deployed App - Git Front-End - Git Back-End

React.js | Node.js | Express | Mongoose

Food Runners was created as a team project where I was responsible for designing the database architecture and the UI design.

The app allows busy food truck operators to receive orders from customers and displays the orders on dashboard for quick preparation and fulfillment.

This specific implementation is for a pizza food truck where customers are able to sign-up for the service and select every ingredient as well as the size of their pizza before submitting to the food truck.

Origin Marketplace

Deployed App - GitHub

JavaScript | HTML | CSS | Node.js | Express | MongoDB

Origin Marketplace allows coffee growers to list their coffee for direct trade in destination markets.

The app implements Google OAuth to authenticate growers, allowing them to add coffees for listing in the marketplace. The marketplace then lists the available coffees for sale.
